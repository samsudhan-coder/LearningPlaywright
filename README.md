# LearningPlaywright

Small demo repo for Playwright experiments and LLM examples.

## Contents

- `01_basics_llm.js` — small example script in this repository.

## Prerequisites

- Node.js (recommended v18+)
- npm or yarn

## Quick start

1. Install dependencies (if you add any):

```bash
npm install
# or
yarn
```

2. Run the example script:

```bash
node 01_basics_llm.js
```
